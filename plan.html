mkdir -p ~/.config/sops/age

grep "public key" ~/.config/sops/age/keys.txt   # copy this

Step 2 — Add public key to .sops.yaml

Add a creation rule for secrets/mba13/*.yaml using the new age public key.

Step 3 — Create secrets file

sops secrets/mba13/default.yaml

  age.keyFile = "${config.home.homeDirectory}/.config/sops/age/keys.txt";
  defaultSopsFile = lib.snowfall.fs.get-file "secrets/mba13/default.yaml";
  secrets = {
    ANTHROPIC_API_KEY  = {};
    OPENCONNECT_USER   = {};
    OPENCONNECT_PW     = {};
    OPENCONNECT_SERVER = {};
    grafana_api_key    = {};
  };
};

GPG Signing — wired ✅ (YubiKey enrolment optional)

Signing is enabled on mba13 via gpg-smart-sign. The fallback path (local key 7C43…, passphrase via pinentry-mac) works today and is validated. Takes effect on the next darwin-rebuild switch.
  • Fallback signing — works now; git commit signs with 7C43… via pinentry-mac.
  • YubiKey path — when the key is plugged in, gpg-smart-sign detects the card (gpg --card-status) and signs with the on-card key 15DB… instead. No config change needed; just plug it in.
  • SSH auth via GPG agent already configured (enable-ssh-support + SSH_AUTH_SOCK).
  • Note: gpg-smart-sign hardcodes both key ids in packages/gpg-smart-sign; they match the current keys. Update there if keys rotate.
